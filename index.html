O que é DIO?

DIO1 é uma plataforma de aprendizado que lhe proporciona oportunidades para aprender e desenvolver conhecimentos em programação e tecnologia começando desde os Primeiros Passos em Tecnologia.

Metodologia:

Na DIO você irá aprender pelas seguintes formas:

  • Vídeo aulas
  • Desafios de código
  • Comunidade
  • Contato com outros devs

Introdução à programação e Pensamento Computacional

O que irá aprender:

É um Bootcamp que lhe ensina os 1os passos em tecnologia. Começando sobre a lógica de programação e seus fundamentos como o que são funções ou variáveis.

Git/Github

O que irá aprender:

O curso lhe ensina sobre o que é e como funciona o Git e o Github. Nesta etapa o estudante aprende sobre o funcionamento do Git e também a criar seu primeiro repositório.

Client x Server

O que irá aprender:

Irá aprender a diferença entre cliente e servidor e entre hardware e software. Também é visto as diferentes linguagens de programação que se aplicam para o lado do cliente, do servidor ou ambos como o JavaScript.

O que é HTML

O que irá aprender:

Não ser confundido com uma linguagem de programação, HTML é usado para escrever o conteúdo de um site ou aplicativo.
